Notification Rules

이메일 알림 대상 컨테이너를 동적으로 관리 — DB에 저장된 규칙은 재시작 없이 즉시 적용됩니다

적용 우선순위
  1. 컨테이너 라벨 notification.enabled=false → 즉시 차단 (모든 규칙 무시)
  2. EXCLUDE 규칙 매칭 → 차단
  3. INCLUDE 규칙이 하나라도 존재 시 화이트리스트 모드 (매칭 안 되면 전부 차단)
  4. 둘 다 없음 → 기본 허용 (단, intentional 종료는 차단)
규칙 추가

등록된 알림 규칙이 없습니다.

규칙이 없으면 모든 비정상 종료에 알림이 발송됩니다 (기본 허용).
