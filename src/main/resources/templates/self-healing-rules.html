Self-Healing Rules

자가치유 규칙을 동적으로 관리 — DB에 저장된 규칙은 재시작 없이 즉시 적용됩니다

자가치유 규칙이란?
  • 컨테이너가 비정상 종료(die/oom/crash)되면 규칙에 맞는 컨테이너를 자동 재시작합니다
  • application.yml의 YAML 규칙 + DB 규칙이 합산되며, 이름 패턴 충돌 시 DB 규칙이 우선합니다
  • maxRestarts 초과 시 재시작 중단 (시간 윈도우 후 리셋)
  • restartDelaySeconds로 지연 재시작 가능 (DB 연결 대기 등)
  • 의도적 종료(docker stop/kill)는 자가치유 대상에서 제외됩니다
규칙 추가

등록된 자가치유 규칙이 없습니다.

규칙이 없으면 application.yml에 정의된 YAML 규칙만 적용됩니다.
