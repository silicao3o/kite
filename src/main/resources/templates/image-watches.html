Image Watches

GHCR 이미지 자동 업데이트 감시 설정 — 새 이미지 감지 시 롤링 업데이트 자동 실행

이미지 와치 원리
  • image: GHCR 이미지 경로 (예: ghcr.io/org/app)
  • tag: 감시할 태그 (기본 latest)
  • containerPattern: 업데이트 대상 컨테이너 이름 패턴 (정규식)
  • maxUnavailable: 동시 업데이트 컨테이너 수 (기본 1 = 순차)
  • GHCR 토큰: 와치별 PAT 설정 가능 (비어있으면 글로벌 설정 폴백)
  • 폴링 주기와 글로벌 GHCR 토큰은 application.yml에서 설정
와치 추가
